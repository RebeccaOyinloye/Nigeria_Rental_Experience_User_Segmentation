### Project Overview
---
This repository contains an analysis of a rental experience dataset, focusing on Nigeria rental experience. The analysis aims to segment users using demographic, psychographic and behavioral segmentation in rental experience, develop insight to inform strategies for improving rental experience in Nigeria.
