# Nigeria_Rental_Experience_User_Segmentation

# Table of content

### Project Overview
---
This repository contains an analysis of a rental experience dataset, focusing on Nigeria rental experience. The analysis aims to segment users of Nigeria rental experience using demographic, psychographic and behavioral segmentation, and develop insight to inform strategies for improving rental experience in Nigeria.

### Data Source
---
The primary source of the data used was from an online survey collected by the team. 
- Note: Some demographic information has been removed to ensure data security and maintain user anonimity

### Segmentation Analysis
---
Role based segmentation 
- Prospective Tenant : Indiviuals renting a property
- Agents: Individuals facilitating rentals
- House Owners/Landlord :Property owners renting out their properties

Geographic Segmentation
- Distribution of regions by state : Analysis of user residency by state and region
- satisfaction rating: Variation in scam rate

Behavioral Segementation 
- Rental status : Recent Renters and Non renters. Users who have rented in Nigeria in the last three years and users who haven't
- preference : willingness to adopt a verified rental platform, bypassing agents/landlord and leasing choices
  
Psychographic Segmentation
- Priorities: Budget Constraints
- Pain Points: Challenges user face in rental process, 
- preference: Users openness to using a platform to facilitate rent
- perception of agent fee

  Demographic segmentation
  - Not included as the dataset did not capture demographic variables such as age, gender or income.
